# دليل التشغيل الحقيقي للبوت (Real Profit Guide)

لقد تم تحديث البوت لإزالة أي بيانات وهمية أو محاكاة. الآن، لن يقوم البوت بعرض أي عملية إلا إذا كانت حقيقية ومربحة فعلياً على الشبكة.

## الخطوات المطلوبة للتشغيل الحقيقي:

### 1. نشر العقد الذكي (Deploy Contract)
يجب نشر العقد الموجود في `contracts/TradingBot.sol` على شبكة BSC (Binance Smart Chain). يمكنك استخدام [Remix IDE](https://remix.ethereum.org) للقيام بذلك:
- قم برفع ملف `TradingBot.sol`.
- اختر نسخة المترجم `0.8.x`.
- قم بنشر العقد واحفظ عنوانه (Contract Address).

### 2. إعداد ملف البيئة (.env)
قم بإنشاء ملف باسم `.env` في المجلد الرئيسي وأضف البيانات التالية:
```env
WALLET_PRIVATE_KEY=مفتاحك_الخاص_هنا
BOT_CONTRACT_ADDRESS=عنوان_العقد_الذي_نشرته_هنا
RPC_URL_BSC=https://binance.llamarpc.com
```

### 3. تمويل البوت
- أرسل كمية من عملة **BNB** إلى عنوان محفظتك (التي وضعت مفتاحها الخاص).
- أرسل كمية من عملة **BNB** إلى عنوان العقد الذكي الذي نشرته (هذه السيولة سيستخدمها البوت للتحكيم).
- الحد الأدنى المقترح للبدء هو **0.05 BNB** لتغطية رسوم الغاز والعمليات الصغيرة.

### 4. التشغيل
قم بتشغيل البوت باستخدام الأوامر التالية:
```bash
npm install
npm run dev
```

## التغييرات التي تم إجراؤها:
1. **إلغاء المحاكاة:** تم حذف الكود الذي كان يقوم بعرض عمليات "وهمية" أو "قريبة من الربح" في الواجهة.
2. **الربط بالعقد الذكي:** البوت الآن يتصل مباشرة بالعقد الذكي الخاص بك لتنفيذ العمليات.
3. **تحقق الربح الصارم:** البوت يتحقق من الربح الحقيقي بعد خصم رسوم الغاز (Gas Fees) قبل محاولة التنفيذ.
4. **دقة البيانات:** تم تحسين جلب الأسعار من PancakeSwap و SushiSwap لضمان دقة الفرص المكتشفة.

**تحذير:** التداول في العملات الرقمية ينطوي على مخاطر. تأكد من تجربة البوت بمبالغ صغيرة أولاً.
